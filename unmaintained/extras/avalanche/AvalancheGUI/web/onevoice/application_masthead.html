HTML/CSS

  • Application mastheads are implemented with a simple table element. The class of these table elements can be either " "applicationMastheadLarge" or "applicationMastheadSmall".
  • The first td element is of class "mastheadIcon", and contains the HP logo image.
  • The second td element is of class "mastheadTitle", and can contain either an H1 tag with your title in Ascii text, or a graphic for your title
  • The third td element is of class "mastheadPhoto", and contains the photographic branding image used by your page.
  • There can be fourth td element, of class "mastheadLinks", which can contain global links and information as shown.

Javascript

  • No extra Javascript is used in this component
