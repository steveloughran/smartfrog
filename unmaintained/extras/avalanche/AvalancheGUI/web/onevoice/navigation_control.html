HTML/CSS

  • The basic element of a Navigation Control item is a link, which exists within a div whose class is either "navigationControlOn" or "navigationControlOff".
  • To avoid CSS width bugs, the immediate parent of a set of navigationControl divs must be a table, and generally this table is then within a tray table or tab component. Here for example it is shown within a div implementing a simple grey-bordered tray .
  • If your application requires that the navigation control manage its own state on the client (ie without being loaded again from the server), wrap the the set of NavigationControl divs within a div of class "NavigationControlSet". When the component is initialized, event handlers will be attached as necessary to maintain state.
  • Note however that this state-managing implementation is currently very basic and does not actually maintain any accessible variables holding that state.

Javascript

  • NavigationControlManager.js contains a method to initialize the components when loaded and a method to manage the highlight state for the State-Managing Navigation Control variation
  • Regarding the state-managing variation, note that this implementation is deliberately a very basic one and does not actually maintain any accessible variables holding that state.
