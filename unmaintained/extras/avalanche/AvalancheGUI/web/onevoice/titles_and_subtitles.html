HTML/CSS

  • Primary titles are implemented as an H1 tag of class "pageTitle". The pagetitle is shown here inside a grey panel, showing also a button element on its right.
  • Secondary titles are implemented as an H3 tag of class "subTitle", followed by a div of class "subTitleBottomEdge". The div is necessary to implement the 3px line in the darker grey color. (A more straightforward bottom-border of 3px cannot be used on the subTitle as this can introduces subtle box-model bugs on IE6.)

Javascript

  • No additional javascript is used for the layout or behaviour of this component, although the ButtonManager is used to set up the button shown
