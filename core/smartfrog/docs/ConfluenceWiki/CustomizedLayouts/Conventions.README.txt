Naming conventions for files in this directory:
 <SPACE>_<TYPE>_<LAYOUTNAME>.txt
 
Where TYPE and LAYOUTNAME are from:

Site
  Main
  Global
  Popup
  Printable
Content
  Space
  Page
  Mail
  Blog
  Classic
  Comments
Export
  SpaceExport
  PageExport
  
E.g.: sf_Content_Page.txt

The .txt extension is used for convenience of editing
