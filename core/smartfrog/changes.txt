2004-Apr-7  Release date
2004-Mar-19 New "smartfrog" shell script
2004-Mar-19 New command entry points for framework
            - ConfigurationDescriptor
            - ActionDeploy
            - ActionTerminate
            - ActionDetachAndTerminate
            - ActionDetach
2004-Mar-17 New utitity methods to
            - create a ComponentDescription from a resource URL:ComponentDescriptionImpl
            - load Parser using a resource URL: SFParser
            - select target process compound: SFProcess
2004-Mar-17 Modified loading of default descriptions in processcompound.
            Now property is: -Dorg.smartfrog.sfcore.processcompound.sfDefault.NAME 
2004-Mar-12 New command line parameters for SFSystem   
2004-Mar-12 Manuals ported to OpenOffice format
2004-Mar-11 New Language operators: 
                Some operators are unary prefix (!) 
                Some are binary (- / >= <= > < == !=)
                Some are nary (+ * && || ++ <>)   (<> is vector append, ++ is string concatenation)
            New: It is now possible to use if-then-else for defining an attribute value
                    x IF (! (foo > 5)) THEN 42 ELSE (foo + 12) FI;
            New: It is now possible to use non-lazy references in the definition of a vector, such as
                    [1,2,x]
2004-Mar-8  Fix: subprocess creation/use race condition 
2004-Mar-3  Add: Normal and Abnormal termination options to Management Console.
2004-Mar-2  New shell scripts for Terminate, DetachAndTerminate, Detach actions
2004-Mar-1  Unit test framework and many tests ported. Available from CVS.
2004-Feb-19 SmartFrog build included in Apache GUMP.
2004-Feb-9  Add: SFNULL reference part for SF language NULL
2004-Feb-11 New parser: JavaCC 3.2. We now need Ant 1.6.1 
2004-Feb-9  Fix: popup windows in MAC-OS X. Thanks to Glenn Hisdal.
