            - create a ComponentDescription from a resource URL:ComponentDescriptionImpl
            - load Parser using a resource URL: SFParser

            Now property is: -Dorg.smartfrog.sfcore.processcompound.sfDefault.NAME

2004-Mar-12 Manuals ported to OpenOffice format
2004-Mar-11 New Language operators: 
                Some operators are unary prefix (!) 
                Some are binary (- / >= <= > < == !=)
                Some are nary (+ * && || ++ <>)   (<> is vector append, ++ is string concatenation)
            New: It is now possible to use if-then-else for defining an attribute value
                    x IF (! (foo > 5)) THEN 42 ELSE (foo + 12) FI;
            New: It is now possible to use non-lazy references in the definition of a vector, such as
                    [1,2,x]
