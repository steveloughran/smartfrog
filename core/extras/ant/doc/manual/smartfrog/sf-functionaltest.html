Sf-functionaltest Task
This is an extension of <junit> that lets us integrate startup, a waitfor condition and a shutdown sequence into the test case.
 
Description
This is an extension of <junit> that lets us integrate startup, a waitfor condition and a shutdown sequence into the test case. This task is used for SmartFrog's internal testing purposes, and for testing components. There are no guarantees of stability over time.

It's based upon Ant's own JUnit task, and credits Cactus as showing what could be done. Apache Cactus is coded around deploying to an application server; this task can deploy to anything for which we have setup and teardown sequences.

This is the workflow for testing

  1. The startup sequence is run to completion
  2. In parallel, ( a) the application is started
  3. and (b) the sequence of waitfor+tests is run.
  4. After the tests run, the teardown operation is executed
  5. Any build exception thrown by testingis thrown

No matter what goes wrong, once startup has succeeded, teardown gets invoked.

 
Parameters
Attribute Description Type Requirement
shutdowntime Set the timeout allowed for a clean shutdown of the application. After this timeout, a more brutal operation is used. int Optional
testtimeout This is the timeout for testing; any timeouts in the probe are not covered under this. int
 
Parameters as nested elements
 
application (org.apache.tools.ant.taskdefs.Sequential)
The sequence of tasks used to define the application. This is for hosting the server in a parallel thread to the test run.
 
probe (org.smartfrog.tools.ant.FaultingWaitForTask)
Add a probe, conditions that have to be met before testing begins.
 
setup (org.apache.tools.ant.taskdefs.Sequential)
Define a sequence of operations to run at startup. After running these, the teardown sequence will be called to tear down the system.
 
teardown (org.apache.tools.ant.taskdefs.Sequential)
A sequence of operations that are used to tear down the system.
 
test (org.apache.tools.ant.taskdefs.Sequential)
Declare a list of junit tasks
