HTML/CSS

  • The table is of class "dataTable"
  • Basic Tables should be titled whenever this is of benefit to the user, and such titles should be implemented with a caption tag. This caption element can however be hidden from screen-based browsers by giving it the classname "displayForAuralBrowsersOnly".
  • the tr for the column header row is of class "captionRow", and must contain only th elements
  • No classname is necessary for the default tablecell appearance, as long as the table is given the correct "dataTable" class.
  • If alternating row colors are necessary, the first data row and alternating rows thereafter should have a class of "altRowColor".
  • If cells in a given column contain only an icon, those cells should be of class "iconCell", to allow for correct padding.
  • If cells in a given column contain only a checkbox or radio button, those cells should be of class "checkboxCell", to allow for correct padding.
  • if cells in a given column contain editable Fields such as dropdown list boxes or textfields, the containing tablecells should be of class "editableFieldCell", to allow for correct padding.

Javascript

  • No javascript is used for the layout or behaviour of the Basic Table.
