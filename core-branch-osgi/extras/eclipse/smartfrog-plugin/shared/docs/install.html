SmartFrog Eclipse Plug-In Installation Instructions

System Requirements

The system requirements for the SmartFrog™ Eclipse Plug-In are the following:

  1. SmartFrog.3.04.006_beta release or later
  2. Java 1.4
  3. Eclipse 3.0 or 3.0.1
  4. The %SFHOME% environment variable must be set to SmartFrog_Install_Dir\dist.
  5. The installation locations must not have any embedded spaces in the path name.

NOTE: This has been tested on the Windows system only.

Installation

To install the SmartFrog Eclipse Plug-In, do the following:

  1. Download the SmartFrog Eclipse Plug-In from https://sourceforge.net/project/showfiles.php?group_id=87384&package_id=132642.

    The SmartFrog_EplugIn_dist_1.0.0.Beta.zip file contains both the org.smartfrog.tools_1.0.0.Beta.zip file (the functionality plug-in) and the org.smartfrog.tools.doc_1.0.0.Beta.zip file (the online help plug-in).
  2. Unzip the files in the %EclipseHome%\eclipse\plugins directory.
