Smartfrog Wrapper

These classes support the SmartFrog as a service under Windows NT or a daemon process in Unix.

The foundation for this -especially Windows support- is the Java Service Wrapper by Tanuki Software. This company encourages users of their (SourceForge hosted) software to send them donations, which you can do via their web page. If you want to know why such a donation is justifiable, take a look at their WrapperManager class, and the native code counterpart, and think how much cross platform development and testing went into it.
