Sf-run Task
Run an application by deploying it locally; only return from ant after it has finished.
 
Description
Run an application by deploying it locally; only return from ant after it has finished. The default timeout of this task is zero; it only exits when finished. It does have a failonerror value of true; any failure to run is an error.
 
Parameters
Attribute Description Type Requirement

inifile Name an ini file can contain data that overrides the basic settings. File
initialsmartfrogfile the name of a smartfrog file to load on startupe File
livenesscheckperiod liveness check in seconds; optional int
livenesscheckretries retry count for liveness checks. int

port port of daemon; optional -default is 3800 Some tasks do not allow this to be set at all. int

spawn sets the spawn flag. makes it hard (no, impossible!) to log outputs. only recommended for long-lived tasks, and complicates failonerror and timeout logic boolean
spawntimeout when to assume a spawn failed -in seconds. int
timeout set the timeout for execution. This is incompatible with spawning. long
