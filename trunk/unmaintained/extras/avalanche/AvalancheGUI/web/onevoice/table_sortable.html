HTML/CSS

  • This is essentially the same component as the basic table, with the following differences:
    • The single sorted TH element is either of class "sortedAscending", or "sortedDescending". This one css class is sufficient to control the appearance of both the darker background and the arrow on the right side of the cell.
    • Other TH elements that are sortable are of class "sortable". This controls the cursor change to the pointing hand.
    • Datacells within the sorted column are to be individually given the class "sorted". This will control the darker color of cells within that column.

Javascript

  • No javascript is specified for the layout or behaviour of this component.
  • Note however that the recommended way to make the column headers clickable is to put onclick handlers into the HTML of each TH element. These onclick handlers should change document.location to return a new view with the appropriate sorting.
