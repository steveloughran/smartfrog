HTML/CSS

  • The grey divider shown is a straightforward HR tag. the global stylesheet styles plain HR tags to appear as 1 pixel thin grey lines.
  • The second divider shown is a span tag of class "whiteSpacer", which contains a single non breaking space element. This tag will implement a vertical gap 10px high, and can be used as an alternative to margin css, to put appropriate vertical spacing between components in a page.

Javascript

  • No javascript is used for the layout or behaviour of this component.
