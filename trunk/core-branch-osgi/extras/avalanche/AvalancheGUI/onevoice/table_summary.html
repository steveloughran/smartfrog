HTML/CSS

  • Summary Tables contain summary cells with right aligned contents. This is implemented with class="numericalCell" placed in the th and all td elements within that column.
  • In addition, they can have a summary row. This is implemented with class="summaryRow" on the tr element.

Javascript

  • No javascript is used for the layout or behaviour of this component.
