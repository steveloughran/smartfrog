Introduction

SmartFrog™ (Smart Framework for Object Groups) is a general-purpose framework for deploying and managing software components based on a specification of their configuration. SmartFrog can be applied to multiple problem domains, though recently the focus has been to use SmartFrog to instantiate complex services and applications on utility infrastructures.
