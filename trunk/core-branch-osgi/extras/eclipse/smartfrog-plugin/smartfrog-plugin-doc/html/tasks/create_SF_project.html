Creating a SmartFrog project

General description

Before you can create SmartFrog description files and Java class component files, you must first create a SmartFrog project. The generated project will contain all the necessary SmartFrog and JRE library JAR files needed to successfully parse and compile the description and Java files.

NOTE: Before proceeding, make sure that the SmartFrog installation location is set in the Eclipse IDE. To check, follow the instructions in the Specifying SmartFrog properties section in the online help.

How to create a SmartFrog project

Purpose

To create a SmartFrog project for your SmartFrog description and Java class component files.

Procedure

  1. From the Eclipse menu bar, select File > New > SmartFrog Project. This opens the following dialog.

  2. Expand SmartFrog and select SmartFrog Project and click Next. Alternatively, you can select the icon from the SmartFrog Eclipse toolbar. Either method brings up the following page where you can specify a project name:

  3. In the Project name: field, specify the project name. Since we are going to use the counter example located in the SmartFrog_Install_Dir\src\org\smartfrog\examples\counter directory, we are naming our project MyCounterProject. Click Next. This brings up the following:

    You can accept all the default values on the various tabs. As you can see from the above screen shot, the relevant libraries have been imported into your project.
  4. Click Finish. You are automatically placed in the SmartFrog perspective.

    The Package Explorer view displays your SmartFrog project populated with the necessary SmartFrog and JRE library JAR files.

Summary

The SmartFrog Eclipse plug-in allows you to easily create a SmartFrog project in which you can create your description and Java component files.
